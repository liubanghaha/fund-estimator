## 二、已完成

### 项目结构（48 个文件）

```
fund-estimator/
├── project.config.json
├── cloudfunctions/           # 6 个云函数，全部已部署
│   ├── createCollection/     # 自动创建数据库集合
│   ├── searchFund/           # 按代码查基金（天天基金 API）
│   ├── fetchFundEstimate/    # 获取实时估值
│   ├── fetchFundInfo/        # 获取基金基本信息
│   └── getPortfolio/         # 聚合持仓收益
└── miniprogram/
    ├── app.js / app.json / app.wxss
    ├── components/fund-card/    # 基金卡片组件
    ├── utils/api.js             # 云函数调用封装
    ├── utils/calculator.js      # 收益计算工具
    └── pages/
        ├── index/           # 首页 — 资产概览
        ├── search/          # 搜索基金（6位代码）
        ├── fund-detail/     # 基金详情 + 涨跌估算
        ├── portfolio/       # 持仓列表 + 汇总统计
        ├── add-holding/     # 添加持仓（买入日期选填）
        ├── login/           # 微信授权登录
        └── user-center/     # 个人中心
```

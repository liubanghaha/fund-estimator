# 小程序开发步骤

## 1. 前期准备

- **注册小程序账号** — 在微信公众平台 (mp.weixin.qq.com) 注册，需实名认证。企业主体需营业执照，个人主体功能受限（如无法使用微信支付）。
- **选择开发工具** — 官方「微信开发者工具」，支持代码编辑、预览、调试、上传。
- **了解技术栈** — 原生框架（WXML + WXSS + JS/TS），或跨端框架 Taro、uni-app。

## 2. 设计与规划

- **产品需求梳理** — 明确核心功能、目标用户、页面结构。
- **UI 设计** — 遵循微信小程序设计规范，设计稿基准 750rpx。
- **后端规划** — 微信云开发（免运维）或自建后端（Node.js/Java/Python 等），需 HTTPS 接口。

## 3. 开发

- **页面开发** — 编写页面、组件、路由配置。
- **接口联调** — 后台配置合法域名（HTTPS + 备案域名），开发阶段可勾选「不校验域名」。
- **登录授权** — 对接微信登录流程（wx.login → code → 服务端换取 openid）。
- **支付接入**（如需要）— 商户号开通 + 统一下单接口对接。

## 4. 测试

- **真机调试** — 必须在真机上测兼容性和性能。
- **体验版测试** — 上传为体验版，添加体验成员，完整流程走一遍。
- **审核自检** — 功能完整可运行、无严重 Bug、类目选择正确、隐私合规。

## 5. 发布与运维

- **提交审核** — 在 MP 后台提交，审核通常 1-7 个工作日。
- **发布上线** — 审核通过后手动发布，用户可搜索和使用。
- **持续运维** — 版本迭代、数据监控、Bug 修复。

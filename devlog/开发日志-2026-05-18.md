# 基金估算小程序 — 开发日志

**日期：** 2026-05-18

---

## 一、新增功能

### 1. 持仓编辑功能
- **add-holding 页**：支持新建和编辑两种模式
  - 编辑模式下基金代码/名称锁定不可改
  - 底部新增「删除此持仓」按钮
- **portfolio 页**：点击任意持仓卡片进入编辑页，「点击编辑」提示

### 2. 实时涨幅与收益计算
- **首页**：顶部大卡片展示今日估算收益金额 + 涨幅百分比，副栏显示市值、累计收益
- **持仓页**：
  - 顶部「今日估算收益」卡片（红涨绿跌）
  - 副栏：持仓成本 / 当前市值 / 累计收益
  - 每只基金卡片：估算净值、今日涨跌幅/收益/市值/累计收益
  - 右上角「↻ 刷新估值」按钮

### 3. 支付宝风格基金详情页（重点）
- **双净值卡片**：
  - 主卡片：最新净值 + 实际日涨跌幅（东方财富数据）
  - 副卡片：盘中估算净值 + 估算涨跌幅（天天基金数据）
- **收益表现**：6 指标网格（日涨幅 / 近1周 / 近1月 / 近3月 / 近6月 / 近1年）
- **历史净值列表**：最近 30 天净值，每日显示日期/净值/涨跌幅
- **底部固定操作栏**：「添加持仓」+「刷新」

### 4. 新增云函数
| 云函数 | 功能 | 数据源 |
|--------|------|--------|
| fetchFundNAVHistory | 历史净值走势 | 东方财富 |
| fetchFundEstimate（升级） | 估值 + 实际涨跌 | 天天基金 + 东方财富 |

- **首页/持仓页**：优先显示 `actualChangeRate`（实际）

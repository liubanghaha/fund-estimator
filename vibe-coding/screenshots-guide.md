# 截图指引

## 截图工具

推荐使用**微信开发者工具**预览并截图，真机截图也可。

微信开发者工具 CLI（已安装）：
```
/Applications/wechatwebdevtools.app/Contents/MacOS/cli auto-preview --project /Users/liubangwe/WeChatProjects/fund-estimator
```

## 必截画面（6 张）

### 1. 首页 — 今日收益卡片 + 持仓列表
- 路径：`pages/index/index`
- 要点：红色渐变收益卡片在顶部，金额在交易时段实时跳动；下方持仓列表展示基金名称、涨跌幅、当日收益
- 构图：竖屏满屏，突出红色的"今日估算收益"数字

### 2. 首页 — 底部指数行情栏展开态
- 路径：同上，点开底部"指数行情"
- 要点：彩色指数卡片阵列展开，红绿分明，展开动画很有质感
- 构图：下半屏，卡片全部可见

### 3. 截图导入 OCR 识别（核心卖点）
- 路径：`pages/add-holding/index`，选择"截图导入"模式
- 要点：展示从支付宝截图 → OCR 识别 → 基金卡片匹配结果的全过程
- 建议：放一张支付宝持仓截图 + 一张识别结果页面的对比图，或直接用 App 内的空态页面展示 `index_demo` 中的演示图
- 示意图可用 pic/index_demo.PNG（支付宝持仓截图）

### 4. 收益详情 — 走势图 + 盈亏日历
- 路径：`pages/profit-detail/index`
- 要点：上半部分收益走势 Canvas 折线图（叠加指数对比），下半部分盈亏日历（GitHub contribution 风格）
- 构图：需要日历区域清晰可见，红绿格子对比明显

### 5. 基金对比页
- 路径：`pages/fund-compare/index`（需传递基金参数）
- 要点：双基金"VS"布局，红蓝归一化走势对比图
- 构图：竖屏，VS 居中，两只基金各占一侧

### 6. 收益走势 — 当天分时图（交易时段截取）
- 路径：`pages/profit-detail/index`，当天视图
- 要点：实时分时走势线 + 指数叠加线，数据点实时更新（每 5 秒）
- 构图：走势图区域放大，显示十字准星和数据标签会更生动

## 可选画面

- 基金详情页（`pages/fund-detail/index`） — 前十大持仓 + 基金经理
- 加减仓 OCR（`pages/adjust-holding/index`）— 交易记录截图识别
- 盈亏日历月视图 / 年视图 — 更大范围的热力图
- 云函数面板 — 展示 18 个云函数列表

## 截图注意事项

- 数据需真实，交易时段截取效果最佳（9:30-11:30, 13:00-15:00）
- 横屏或竖屏统一，建议全部竖屏 9:16
- 金额数字如需隐藏，点首页金额可切换显示/隐藏
- 截图命名：`01-home.png`, `02-index-bar.png`, `03-ocr.png`, `04-profit.png`, `05-compare.png`, `06-intraday.png`

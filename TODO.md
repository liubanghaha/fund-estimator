## 💰 变现与增长

> 基于当前 2,326 用户 / 1,000 DAU，优先通过会员制变现（0 审核风险）。
> 目标：5% 付费转化 + 社交传播增长，3 个月内 DAU 翻倍。

### 🔥 第一阶段：直接变现（Pro 会员）

- [ ] **Pro 会员订阅** — ¥9.9/月 or ¥68/年
  - 免费版：近 1 年历史数据、定投回测 3 次/月、沪深300 对比、30s 刷新
  - Pro 版：近 5 年历史数据、定投回测无限次、自选 3 指数对比、10s 刷新
  - 云函数新增 `userSubscribe`（支付回调 + 会员状态管理）
  - DB 新增 `subscriptions` 集合：`{ _openid, plan, startDate, endDate, status }`
- [ ] **数据导出** — ¥3.9/次 or Pro 免费
  - CSV 导出：交易记录、持仓明细、收益日历
  - 审核零风险（纯数据导出）
- [ ] **年度投资报告** — ¥3.9 一次性 or Pro 免费
  - Canvas 绘制全年收益走势 + 费用汇总 + 调仓回顾
  - 可保存相册、分享朋友圈（带小程序码引流）
  - 文案避开"收益报告""投资建议"，用"持仓回顾""年度总结"等中性词

### 📈 第二阶段：增长驱动（免费功能，间接变现）

- [ ] **收益卡片分享** — 一键生成精美持仓卡片
  - 模板：今日收益 + 本月收益率 + 跑赢指数 + 小程序码
  - Canvas 绘制，保存到相册
  - 分享到微信群/朋友圈，最便宜的拉新手段
- [ ] **投资笔记** — 加减仓时可选记理由
  - 事后回看追溯决策逻辑，高迁移成本 → 强留存
  - 纯个人数据管理，审核零风险
- [ ] **自定义提醒** — 扩展止盈止损，新增数据驱动通知
  - 净值回撤超 X% 提醒
  - PE 温度档位变化提醒（如"合理 → 高估"）
  - 季报发布提醒
  - 全部基于数据通知，不构成投资建议

# v1.0.0 版本描述

## 核心功能
本小程序为基金持仓管理工具，帮助用户追踪基金投资收益。

## 主要功能
1. **持仓管理**：截图导入或手动添加基金持仓，自动识别基金信息
2. **实时估值**：展示持仓基金实时估算净值和涨跌幅
3. **收益走势**：本周/本月/本年收益走势图，支持沪深300等主要指数对比
4. **加减仓**：支持加仓和减仓操作，自动更新持仓份额和成本
5. **自选列表**：添加自选基金，查看估值变化
6. **基金对比**：两只基金收益走势和基本信息对比
7. **交易记录**：查看每笔加仓/减仓的历史记录

## 数据说明
- 基金净值数据来源于公开市场数据
- 收益计算基于用户手动录入的持仓信息
- 本工具仅供个人参考，不构成投资建议
